#  GoldenHub CRM — Business Website
GoldenHub CRM is a fictional Customer Relationship Management (CRM) platform designed for sales and marketing teams in South Africa. This project involved planning, designing, building, testing, and documenting a complete 4-page business website using only HTML and CSS 

The website was inspired by real CRM products like HubSpot, Zoho, and Monday.com, but designed to be simpler and more affordable for small and medium-sized African businesses.

---

Live Website

 [View the Live Site on GitHub Pages](https://goldenhubcrm.netlify.app/)


 Pages

|  Home | `index.html` | Hero section, features, how it works, stats and CTA |
|  About | `about.html` | Company story, mission, values and team |
|  Services | `services.html` | Service offerings, pricing plans and comparison |
|  Contact | `contact.html` | Contact form, contact info and FAQ section |

##  Design Choices

### Colour Palette

| Colour | Hex Code  Used For|

| Primary Blue | `#2a7fc1` | Buttons, headings, links, card borders |
| Gold Accent | `#f5a623` | Logo, highlights, hover effects |
| Dark Navy | `#1a1a2e` | Page banners, footer background |
| Light Blue | `#e8f4fd` | Alternating section backgrounds |
| White | `#ffffff` | Main page background, cards |
| Dark Grey | `#333333` | Body text and paragraphs |

### Typography
- **Headings and Navigation** — Arial, Helvetica (sans-serif)
- **Body Text** — Georgia (serif)

Both fonts are web-safe and load on all devices without external dependencies.
